## 💡 策略特点

### 1️⃣ 选币策略
- **空单**：选择涨幅前6的币种做空（做空强势币）
- **多单**：选择涨幅前6的币种做多（追涨强势币）

### 2️⃣ 仓位管理
| 策略类型 | 单个币种仓位 | 总仓位（6个币） | 杠杆后总曝险 |
|----------|--------------|-----------------|--------------|
| **空单** | 3% | 18% | 180%（10倍杠杆） |
| **多单** | 8% | 48% | 480%（10倍杠杆） |

### 3️⃣ 风险控制
- **空单止损**：-30%
- **多单止损**：-20%
- **保证金预留**：每个策略预留对应仓位比例的保证金

### 4️⃣ 分批止盈
- 每个策略都采用分批止盈
- 空单策略A最激进（平仓比例最高）
- 空单策略C最保守（平仓比例最低）
- 多单策略A/B/C的平仓比例逐步递增

## 🚀 使用说明

### 1. 启用/禁用策略
修改配置文件中的 `"enabled"` 字段：
```json
{
  "空单策略A": {
    "enabled": true  // true=启用, false=禁用
  }
}
```

### 2. 调整仓位
修改 `risk_management.max_position_percent` 字段：
```json
{
  "risk_management": {
    "max_position_percent": 3  // 单个币种的仓位比例
  }
}
```

### 3. 调整选币数量
修改 `rank_based_selection.top_n` 字段：
```json
{
  "rank_based_selection": {
    "top_n": 6  // 选择排名前N的币种
  }
}
```

### 4. 调整杠杆
修改 `leverage` 字段：
```json
{
  "leverage": 10  // 杠杆倍数
}
```
