# 全屏大气加载页面设计文档

## 🎨 设计理念

从小气的内联进度条升级为**全屏沉浸式加载体验**，营造专业、现代的视觉氛围。

## 📐 布局结构

### 1. 全屏覆盖层
- **尺寸**: 100vw × 100vh（覆盖整个视口）
- **定位**: fixed，z-index: 9999（最顶层）
- **背景**: 深色渐变（#0a0e27 → #1a1a2e → #16213e）

### 2. 背景动画粒子
- **数量**: 5个彩色粒子
- **颜色**: 
  - 绿色 `rgba(0, 255, 136, 0.6)`
  - 青色 `rgba(0, 204, 255, 0.6)`
  - 紫色 `rgba(102, 126, 234, 0.6)`
  - 橙色 `rgba(245, 158, 11, 0.6)`
  - 红色 `rgba(239, 68, 68, 0.6)`
- **动画**: float（15~22秒，无限循环）
- **效果**: 粒子在屏幕中缓慢飘动

## 🎬 动画效果

### 1. float（粒子飘动）
```css
0%, 100% → (0, 0)
25%      → (30px, -30px)
50%      → (-20px, 20px)
75%      → (20px, 30px)
```
**时长**: 15~22秒（不同粒子）  
**循环**: 无限

### 2. pulse（Logo脉动）
```css
0%, 100% → scale(1)
50%      → scale(1.05)
```
**时长**: 2秒  
**循环**: 无限

### 3. ripple（光圈扩散）
```css
0%   → scale(1), opacity: 1
100% → scale(1.3), opacity: 0
```
**时长**: 2秒  
**循环**: 无限  
**延迟**: 0s / 0.5s（两个光圈）

### 4. slideDown（标题下滑）
```css
from → translateY(-30px), opacity: 0
to   → translateY(0), opacity: 1
```
**时长**: 0.8秒  
**延迟**: 0s / 0.2s（主标题/副标题）

### 5. shine（进度条光泽）
```css
0%   → left: -100%
100% → left: 200%
```
**时长**: 2秒  
**循环**: 无限

### 6. bounce（圆点弹跳）
```css
0%, 80%, 100% → scale(0), opacity: 0.5
40%           → scale(1), opacity: 1
```
**时长**: 1.4秒  
**循环**: 无限  
**延迟**: 0s / 0.2s / 0.4s

### 7. fadeOut（完成淡出）
```css
opacity: 1 → 0 (500ms)
```
**触发**: 进度达到100%  
**延迟**: 800ms  
**时长**: 500ms

## 🎯 交互逻辑

### 加载流程
1. **显示**: 页面加载时 `display: block`
2. **更新**: 调用 `updateLoadProgress(step, total, text)`
   - 更新进度条宽度
   - 更新百分比数字
   - 更新进度文本
3. **完成**: 100%后延迟800ms → 淡出500ms → 隐藏

### JavaScript接口
```javascript
updateLoadProgress(step, total, text)
```

**参数**:
- `step`: 当前步骤（1, 2, ...）
- `total`: 总步骤数（如2）
- `text`: 描述文字（如"正在加载2026-02-15数据..."）

**效果**:
- 进度条宽度: `(step / total) * 100%`
- 百分比: `Math.round((step / total) * 100) + '%'`
- 文本: `${text} (${step}/${total})`
