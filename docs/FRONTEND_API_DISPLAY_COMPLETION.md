## 🔧 技术实现

### JavaScript函数

```javascript
// 核心函数列表
loadHome()              // 加载首页数据
loadApiList()          // 加载接口列表
renderApiList()        // 渲染接口展示
filterApiList()        // 搜索过滤接口
```

### 数据流程

```
1. 页面加载
   └── loadHome()
       └── fetch('/api/data-sync/sender/catalog')
           └── loadApiList(catalogData)
               └── 解析分类和端点
                   └── renderApiList(allApiEndpoints)

2. 搜索过滤
   └── 用户输入关键词
       └── filterApiList()
           └── 过滤 allApiEndpoints
               └── renderApiList(filtered)
```

## 📖 使用说明

### 访问管理界面

```bash
https://your-domain.com/data-sync-manager
```

### 查看接口列表

1. 打开管理界面
2. 默认显示"首页"标签
3. 向下滚动到"📡 数据端点接口展示"区域
4. 查看所有30个接口的详细信息

### 使用搜索功能

1. 在搜索框输入关键词
2. 实时过滤显示匹配的接口
3. 清空搜索框显示所有接口

### 复制使用示例

每个接口都提供了curl命令示例：

```bash
curl -H "X-Request-From: https://your-system.com" \
     -H "X-System-ID: your_system_id" \
     https://sender.com/api/data-sync/sender/data/{CODE}
```

只需替换：
- `https://your-system.com` → 您的系统主网址
- `your_system_id` → 您的系统ID
- `https://sender.com` → 发送端主网址
- `{CODE}` → 端点编码（如PT001）
