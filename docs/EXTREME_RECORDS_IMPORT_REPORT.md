
### 7.4 按币种统计
```sql
SELECT 
    inst_id,
    COUNT(*) as total,
    MAX(CASE WHEN record_type = 'max_profit' THEN profit_rate END) as max_profit,
    MIN(CASE WHEN record_type = 'max_loss' THEN profit_rate END) as max_loss
FROM position_extreme_records
GROUP BY inst_id
ORDER BY total DESC;
```
