# 全屏加载页面设计说明

## 🎨 设计概览

全新的全屏加载页面采用现代化、大气的设计风格，提供沉浸式的加载体验。

## 📐 布局结构

### 1. 全屏覆盖层
- **尺寸**：100vw × 100vh（完全覆盖整个视口）
- **层级**：z-index: 9999（最高层级，遮盖所有内容）
- **定位**：fixed（固定定位，不随页面滚动）

### 2. 渐变背景
```css
background: linear-gradient(135deg, #0a0e27 0%, #1a1a2e 50%, #16213e 100%)
```
- **起点颜色**：#0a0e27（深蓝黑）
- **中间颜色**：#1a1a2e（深灰蓝）
- **终点颜色**：#16213e（中度蓝）
- **方向**：135度对角渐变

## 🎬 动画关键帧

### float（粒子浮动）
```css
0%, 100%: translate(0, 0)
25%: translate(30px, -30px)
50%: translate(-20px, 20px)
75%: translate(20px, 30px)
```

### pulse（Logo脉冲）
```css
0%, 100%: scale(1)
50%: scale(1.05)
```

### ripple（光圈扩散）
```css
0%: scale(1), opacity: 1
100%: scale(1.3), opacity: 0
```

### slideDown（文字下滑）
```css
from: translateY(-30px), opacity: 0
to: translateY(0), opacity: 1
```

### shine（进度条光泽）
```css
0%: left: -100%
100%: left: 200%
```

### bounce（圆点弹跳）
```css
0%, 80%, 100%: scale(0), opacity: 0.5
40%: scale(1), opacity: 1
```

## 🔄 加载流程

### 显示阶段
1. **初始状态**：display: none
2. **触发显示**：updateLoadProgress() 调用时设为 display: block
3. **进度更新**：
   - progressBar 宽度从 0% → 100%
   - progressPercentage 文本更新（0% → 100%）
   - progressText 显示当前步骤信息

### 完成阶段
1. **达到100%**后等待 800ms
2. **淡出动画**：
   - 添加 transition: opacity 0.5s ease-out
   - opacity: 1 → 0（500ms）
3. **隐藏**：display: none
4. **重置透明度**：opacity: 1（为下次显示准备）

## 🔧 技术实现

### HTML结构
```html
<div id="loadingProgress">
  └─ 背景粒子层
  └─ 主内容区
      ├─ Logo区（带光圈）
      ├─ 标题和副标题
      ├─ 进度条容器
      │   ├─ 百分比显示
      │   ├─ 进度条
      │   └─ 文本说明
      ├─ 加载动画点
      └─ 版本信息
  └─ CSS动画样式
</div>
```

### JavaScript API
```javascript
updateLoadProgress(step, total, text)
// step: 当前步骤 (1-10)
// total: 总步骤数 (10)
// text: 步骤描述文本
```
