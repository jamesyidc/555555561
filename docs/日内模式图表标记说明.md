# 日内模式检测在总涨跌幅趋势图中的标记显示

## 功能概述

在总涨跌幅趋势图中，日内模式检测的触发事件会以**显眼的标记**显示，区别于其他指标（市场情绪信号、RSI等）。

## 标记样式设计

### 三种信号类型

1. **🔻 做空信号**（红色倒三角）
   - 图标：倒置三角形
   - 颜色：亮红色 (#EF4444)
   - 位置：数据点上方
   - 触发条件：操作建议包含"做空"或"新低"

2. **🚀 做多信号**（绿色箭头）
   - 图标：向上箭头
   - 颜色：亮绿色 (#10B981)
   - 位置：数据点下方
   - 触发条件：操作建议包含"做多"、"低吸"或"抄底"

3. **⏸️ 观望信号**（橙色钻石）
   - 图标：钻石形
   - 颜色：橙色 (#F59E0B)
   - 位置：数据点上方
   - 触发条件：操作建议包含"观望"或"等待"

### 视觉特性

- **图标大小**：35px（比市场情绪信号的24px更大）
- **边框效果**：3px白色边框
- **阴影效果**：
  - 模糊半径：10px
  - 阴影偏移：Y轴3px
  - 颜色：半透明（rgba，透明度0.6）
- **标签样式**：
  - 字号：14px，加粗
  - 白色文字 + 深色背景
  - 圆角：6px
  - 内边距：6px 12px
  - 带阴影效果

### 图层层级

- **zlevel: 15** - 最高层级
- 市场情绪信号：zlevel 10
- RSI标记：默认层级
- 波峰标记：默认层级

## 标记位置计算

### 时间范围解析

模式检测记录包含时间范围（例如："06:30 - 06:50"），系统会：

1. 解析起始时间和结束时间
2. 在图表时间轴中查找对应的索引
3. 使用**中间点**的索引和涨跌幅作为标记位置

### 示例

```
时间范围: 06:30 - 06:50
├─ 起始时间: 06:30 → 索引: 300
├─ 结束时间: 06:50 → 索引: 330
└─ 标记位置: (300 + 330) / 2 = 315
```

## 数据加载流程

1. **API请求**
   ```
   GET /api/intraday-patterns/detections/<date>
   ```

2. **数据处理**
   - 解析时间范围
   - 匹配图表时间索引
   - 分类信号类型
   - 生成标记点数据

3. **图表渲染**
   - 作为独立的scatter系列
   - 使用ECharts的数据驱动渲染
   - 响应式更新
