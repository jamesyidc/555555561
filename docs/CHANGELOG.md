# Changelog - 极值追踪系统

## [v1.2.0] - 2026-01-19

### 🎉 新增功能
- **27个币种完整价格信息展示**: 在极值追踪页面显示每个币种的当前价格、基准价格和涨跌幅

## [v1.0.0] - 2026-01-17

### 🎉 初始版本
- **极值追踪系统**: 自动记录极值时27币快照
- **5种触发条件**:
  1. 逃顶信号2h出现预警标记 (>50)
  2. 27种币涨跌幅相加超过100%
  3. 27种币涨跌幅相加小于-80%
  4. 逃顶24h出现极值被标记 (>200)
  5. 1小时爆仓金额超过3000万美元
- **数据存储**: JSON Lines格式，便于扩展和查询
- **API接口**: 提供统计和快照查询接口
