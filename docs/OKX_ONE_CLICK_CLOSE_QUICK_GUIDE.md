# OKX"一键全平"快速使用指南

**功能**: 一键平掉所有账户的所有持仓（多单+空单）  
**位置**: OKX交易页面 → 账户信息区 → "⚙️ 管理账户"按钮旁边  
**按钮**: 🚨 一键全平（红色醒目按钮）

---

## 🚀 快速使用

### 步骤1: 打开交易页面
访问: https://5000-idfgz76cf9poiqtgzfhan-c81df28e.sandbox.novita.ai/okx-trading

### 步骤2: 点击按钮
在页面顶部账户信息栏找到 **🚨 一键全平** 按钮（红色）

### 步骤3: 确认操作
系统会弹出确认框，显示：
- 将要操作的所有账户名称
- 警告提示：将平掉所有多单和空单

点击"确定"继续，或"取消"放弃

### 步骤4: 等待完成
系统会：
1. 自动获取所有账户的持仓
2. 逐个平掉每个持仓（间隔200ms）
3. 显示详细的操作结果

### 步骤5: 查看结果
操作完成后显示：
- ✅ 成功平仓数量
- ❌ 失败数量
- 📊 每个账户的详细情况

---

## ⚠️ 重要提示

1. **不可撤销**: 点击确认后立即开始平仓
2. **全账户操作**: 会平掉**所有**账户的**所有**持仓
3. **市价平仓**: 使用市价单，可能有滑点
4. **需要时间**: 大量持仓可能需要数十秒
5. **API凭证**: 账户需配置完整的API Key、API Secret、Passphrase

## 🔧 故障排查

### 问题: 提示"没有找到任何账户"
**解决**: 
1. 点击"⚙️ 管理账户"
2. 添加至少一个账户
3. 配置API凭证

### 问题: 提示"未配置API凭证"
**解决**: 
1. 打开账户管理
2. 为账户添加：
   - API Key
   - API Secret
   - Passphrase

### 问题: 部分持仓平仓失败
**原因**: 
- API速率限制
- 网络问题
- 持仓已被手动平掉
- 交易对暂停交易

**解决**: 查看详细错误信息，手动处理失败的持仓
