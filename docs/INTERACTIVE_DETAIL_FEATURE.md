## 🎯 使用流程

1. 在页面顶部的日期选择器中选择日期（1月3日-1月16日）

- **27币涨跌幅总和**: 如 +5.23% 或 -3.45%

## 📊 数据规格

### 价格精度
- **基准价格**: 8位小数 ($95,392.80000000)
- **当前价格**: 8位小数 ($95,500.12345678)
- **涨跌幅**: 4位小数 (+0.1125%)

### 币种顺序
按市值和重要性排序：
```
BTC, ETH, XRP, BNB, SOL, LTC, DOGE, SUI, TRX, TON,
ETC, BCH, HBAR, XLM, FIL, LINK, CRO, DOT, UNI, NEAR,
APT, CFX, CRV, STX, LDO, TAO, AAVE
```

### 计算公式
```javascript
// 单个币种涨跌幅
涨跌幅 = (当前价格 - 基准价格) / 基准价格 × 100%

// 27币总和
总和 = Σ(BTC涨跌幅 + ETH涨跌幅 + ... + AAVE涨跌幅)
```
