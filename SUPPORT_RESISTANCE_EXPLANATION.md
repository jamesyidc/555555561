## 💡 这个目录存储了什么数据？

### 核心功能
这个目录存储**27个加密货币的支撑压力位数据**，用于判断币价在支撑和压力之间的位置。

### 支撑压力位解释

```
              ┌─────────────────┐  压力线1 (7天最高价)
              │                 │  ← 币价难以突破的上方阻力
              │                 │
              │   ● 当前价格    │  
              │                 │
              │                 │
              └─────────────────┘  支撑线1 (7天最低价)
                                   ← 币价难以跌破的下方支撑
```

### 两种周期
1. **7天周期**: 支撑线1 和 压力线1 (基于7天最高/最低价)
2. **48小时周期**: 支撑线2 和 压力线2 (基于48小时最高/最低价)

## 🚨 4种预警场景

### 场景1: 接近48小时支撑位 ✅ 买入机会
- **触发条件**: `position_48h ≤ 5%`
- **含义**: 币价跌到近48小时的最低点附近
- **操作建议**: 可能是短线买入机会（可能反弹）

### 场景2: 接近7天支撑位 ✅ 强买入信号
- **触发条件**: `position_7d ≤ 5%`
- **含义**: 币价跌到近7天的最低点附近
- **操作建议**: 较强的买入机会（支撑力度更强）

### 场景3: 接近48小时压力位 ⚠️ 卖出警告
- **触发条件**: `position_48h ≥ 95%`
- **含义**: 币价涨到近48小时的最高点附近
- **操作建议**: 可能是短线卖出时机（可能回调）

### 场景4: 接近7天压力位 ⚠️ 强卖出信号
- **触发条件**: `position_7d ≥ 95%`
- **含义**: 币价涨到近7天的最高点附近
- **操作建议**: 较强的卖出时机（压力较大）

## 📈 位置百分比计算公式

### position_7d (7天位置)
```
position_7d = (当前价格 - 支撑线1) / (压力线1 - 支撑线1) × 100%

例: ETC当前价格 $11.98
    支撑线1 = $11.83
    压力线1 = $12.49
    
    position_7d = (11.98 - 11.83) / (12.49 - 11.83) × 100%
                = 0.15 / 0.66 × 100%
                = 22.73%
```

**解读**: 币价位于7天支撑压力区间的22.73%位置（偏下方，接近支撑）

### position_48h (48小时位置)
```
position_48h = (当前价格 - 支撑线2) / (压力线2 - 支撑线2) × 100%

例: ETC当前价格 $11.98
    支撑线2 = $11.83
    压力线2 = $12.07
    
    position_48h = (11.98 - 11.83) / (12.07 - 11.83) × 100%
                 = 0.15 / 0.24 × 100%
                 = 62.50%
```

**解读**: 币价位于48小时支撑压力区间的62.50%位置（中间偏上）

## 🎯 实战应用场景

### 场景A: 抄底机会
```
币种: BTC
position_7d: 3.2%  ← ≤ 5%, 触发场景2

操作: 🟢 强买入信号，可能是抄底机会
风险: 如果跌破支撑位，可能继续下跌
```

### 场景B: 见顶信号
```
币种: ETH
position_7d: 96.8%  ← ≥ 95%, 触发场景4
position_48h: 97.2% ← ≥ 95%, 触发场景3

分析: 双重压力位预警，币价在7天和48小时的最高点附近
操作: 🔴 强卖出信号，可能面临回调
风险: 如果突破压力位，可能继续上涨
```

```

操作: ⚪ 观望，等待接近支撑或压力位再操作
```

```

示例:
- support_resistance_20260207.jsonl  ← 2026年2月7日
```
