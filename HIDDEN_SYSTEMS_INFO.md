**重要**: 虽然这些系统在数据管理界面中被隐藏，但**实际数据文件仍然保留**在磁盘上：


1. 历史数据可用于回测和分析

## 📝 如何恢复显示

如果需要重新显示这些系统，可以：

### 方法1: 修改配置文件

编辑 `generate_system_grouped_data.py`，将对应系统配置添加回 `SYSTEM_MAPPING`：

```python
SYSTEM_MAPPING = {
    # ... 其他系统 ...
    "支撑压力(大盘)": {
        "dirs": ["support_resistance_jsonl", "support_resistance_daily"],
        "icon": "🎯",
        "color": "#3B82F6"
    },
    "逃顶信号系统": {
        "dirs": ["escape_signal_jsonl"],
        "icon": "🚨",
        "color": "#EC4899"
    }
}
```

### 方法2: 重新生成数据

运行以下命令重新扫描：

```bash
cd /home/user/webapp
python3 generate_system_grouped_data.py
```

### 方法3: 访问数据管理API

访问数据管理页面，点击"扫描数据"按钮，系统会自动重新生成分组数据。
